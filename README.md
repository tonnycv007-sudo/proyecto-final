# VetCare – Sistema de Gestión Veterinaria

VetCare es un sistema desarrollado con Laravel para la administración integral de una clínica veterinaria.  
Incluye herramientas y módulos diseñados para facilitar el control operativo y administrativo del personal, roles, permisos y mascotas.

---

## Características principales

- Gestión de roles y permisos mediante Spatie Permission.
- Administración completa de usuarios del sistema.
- Módulo de mascotas (Pets) con CRUD completo.
- Tablas dinámicas con Livewire y Rappasoft Datatables.
- Interfaz moderna usando WireUI, TailwindCSS y Flowbite.
- Alertas interactivas mediante SweetAlert2.
- Arquitectura limpia, escalable y fácil de mantener.

---

## Tecnologías utilizadas

- Laravel 12  
- Livewire 3  
- WireUI  
- TailwindCSS  
- Flowbite  
- Rappasoft Livewire Datatables  
- SweetAlert2  
- MySQL  
- Spatie Roles & Permissions
